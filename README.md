### babel
babel编译最新版Javascript
```
npm run babel
```
运行编译后js文件
```
node dist/index.js
```

### typescript
#### npm run
使用babel转换typescript为js文件。
```
npm run tsc
```
运行编译后js文件
```
node dist/index.js
```

#### webpack
使用webpack转换typescript
```
webpack
```
运行编译后js文件
```
node dist/app.bundle.js
```

**NOTE**：提交时会自动检查eslint和prettier错误，需要根据提示错误修改代码后再次提交。
